# L7 Objects Activity - Fantastic Beasts Sanctuary

A simple web page to keep track of fantastic beasts. You can add creatures, delete them, and search for them.

* Ayush Patel - (aspatel@dal.ca)


## Built With
* HTML5, CSS, JavaScript and Bootstrap 4.3.1

## What This Does and how it works:
1. Add Creatures - Fill in the form and click Add Creature
2. Delete Creatures - Click the Delete button next to any creature
3. Search - Type in the search box to find creatures by name or type
