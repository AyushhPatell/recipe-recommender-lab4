# Lecture 9: Event Listeners Activity

A registration form that demonstrates the use of JavaScript event listeners to change input field styling dynamically.

* Date Created: 22 OCT 2025
* Last Modification Date: 22 OCT 2025
* Activity URL (Timberlea): https://web.cs.dal.ca/~aspatel/csci3172/activities/lecture9/
* GitLab URL: https://git.cs.dal.ca/aspatel/csci3172/-/tree/main/activities/lecture9

## Author
* Ayush Patel (ayush@dal.ca)

## Files Included
- index.html - Registration form with Bootstrap styling
- css/style.css - Custom CSS including the 'active' class for focused inputs
- js/script.js - JavaScript file with focus/blur event listeners

## Features Implemented
- Event listeners for 'focus' event on First Name, Last Name, and Email fields
- Event listeners for 'blur' event on First Name, Last Name, and Email fields

## Built With
* HTML, CSS, Bootstrap, and JavaScript.
