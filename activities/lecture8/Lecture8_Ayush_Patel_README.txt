# L8 Calculator Activity - Simple 4-Operation Calculator

A basic calculator that can add, subtract, multiply, and divide two numbers. It also shows a greeting based on the current time of day.

* Date Created: 14 Oct 2025
* Last Modification Date: 14 Oct 2025

* GitLab URL: https://git.cs.dal.ca/aspatel/csci3172/-/tree/main/activities/lecture8


## Author
* Ayush Patel - (aspatel@dal.ca)

## Built With
* HTML5, CSS, JavaScript and Bootstrap 4.3.1

## Features
1. Time based Greeting.
2. Addition.
3. Subtraction.
4. Multiplication.
5. Division.
6. Four separate buttons for each operation
