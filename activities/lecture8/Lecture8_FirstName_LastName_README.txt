# L8 Calculator Activity - Simple 4-Operation Calculator

A basic calculator that can add, subtract, multiply, and divide two numbers.
It also shows a greeting based on the current time of day.


* Timberlea URL: https://web.cs.dal.ca/~aspatel/csci3172/activities/lecture8/
* GitLab URL: https://git.cs.dal.ca/aspatel/csci3172/-/tree/main/activities/lecture8

## Features

1. Time based Greeting.
2. Addition.
3. Subtraction.
4. Multiplication.
5. Division.
6. Four separate buttons for each operation

## Learnings from:
* L8 lecture for the starter code and DOM examples
* Course materials for JavaScript basics
