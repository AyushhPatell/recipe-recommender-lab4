# Lab 2: TechShop Inventory Management System

A simple inventory management system for an electronics and computer parts store built using JavaScript, HTML, and CSS.

* Git URL: https://git.cs.dal.ca/aspatel/csci3172/-/tree/main/Labs/Lab1
* Timberlea URL: https://web.cs.dal.ca/~aspatel/csci3172/labs/lab2/


## Features Implemented

- 10 inventory items (electronics and computer parts).
- Items stored in an array with properties: name, type, price, quantity, description.
- Adds new items to inventory.
- Removes items by name.
- Displays all inventory items.
- Searches items by name or type.
- Calculates total inventory value.
- Groups items by type/category.
- Finds duplicate items.
- Applies percentage discount to all items.

## Files Included
- index.html - Main HTML page
- css/styles.css - Stylesheet for page styling
- js/script.js - JavaScript file with all inventory functions
